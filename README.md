## 📖 About the Project

This is a **fully hand-crafted, single-file HTML blog** designed to
explain Generative AI in a clear, structured, and visually engaging way.

It covers everything from: - Basic concepts\
- Core technologies\
- Real-world applications\
- Future trends\
- Learning roadmap

------------------------------------------------------------------------

## 🧠 What You'll Learn

-   What is Generative AI\
-   How Generative AI works\
-   Key technologies: LLMs, GANs, Diffusion Models\
-   Real-world applications\
-   Advantages & challenges\
-   Ethical considerations\
-   Future of AI\
-   Learning roadmap + project ideas
